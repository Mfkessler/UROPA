For developping UROPA the following versions were used : 
Make sure all prerequisites are met:

* [R/Rscript](http://www.r-project.org/) , v3.3.0
* [python](http://continuum.io/downloads) , v2.7.8-anaconda-2.1.0

Packages required :

For python :

- argparse
- logging
- subprocess
- multiprocessing
- itertools

For R:

- library(VennDiagram)
- library(ggplot2)
- library(jsonlite)
- library(gridExtra)
- library(gplots)
- library(Vennerable)
