* [R/Rscript](http://www.r-project.org/)
* [python](http://continuum.io/downloads)

* Python, version used for developping UROPA : 2.7.8-anaconda-2.1.0

- logging
- subprocess
- multiprocessing
- itertools

*R, version used : 3.3.0
- library(VennDiagram)
